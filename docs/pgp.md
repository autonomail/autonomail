# PGP

## Key strength

All keys are RSA-2096 or stronger. On a Macbook Air 2012 model it take roughly the following item to generate keys of the different bit-sizes:

* 2048 bits - 2.5 minutes
* 4096 bits - 20 minutes

Thus we give the user 3 choices:

* Import an existing PGP keypair
* Create a 2048-bit keypair
* Create a 4096-bit keypair
